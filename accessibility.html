Accessibility commitment

Accessibility

We want this website to be usable for more people across devices, languages, and browsing contexts.

Readable structure

We use clear headings, mobile-friendly layouts, visible focus states, and consistent navigation patterns.

Language access

The site is being maintained in both English and Spanish so core information is easier to access across the local community.

Ongoing improvement

Accessibility is not finished in one pass. We continue improving contrast, interactions, labels, and content clarity.

Current focus

What we are actively trying to do well

We are working toward cleaner mobile layouts, clearer action labels, more accurate alt text, less misleading visual affordance, and better form guidance.

If you encounter a barrier, a confusing interaction, or content that does not work with your assistive technology, please tell us so it can be improved.
